AI Receptionist Widget

A production-grade embeddable chatbot widget. Click the button in the bottom-right corner to open it. The mock API simulates real responses.

Embed on any website — one script tag

<script src="https://cdn.linor.ai/widget.js" data-api-key="sk-your-key" data-api-url="https://api.linor.ai/v1/chat" data-bot-name="Aria" data-primary-color="#6366f1" data-position="bottom-right" data-offset-x="24" data-offset-y="24" async ></script>

Programmatic control (after script loads)

window.LinorWidget.open(); window.LinorWidget.close(); window.LinorWidget.toggle(); window.LinorWidget.destroy(); window.LinorWidget.on('send', (text) => console.log(text));

Features

Shadow DOM isolation — zero CSS leakage
4 positions: bottom/top × left/right
window.LinorConfig + data-* attributes
window.LinorWidget public JS API
Session persistence (localStorage)
Retry logic with exponential backoff
Full keyboard navigation + ARIA
Mobile-first responsive design
Typing indicator & loading states
Error states with per-message retry
Configurable size, offset, z-index
Optional attribution hide
